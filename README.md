<a id="testing"></a>

## 🧪 Model Testing

<br>

### 📊 Scoring

```
Score = Base × TimeFactor × CorrectnessCoefficient
  Base:        5 pts/question
  TimeFactor:  ≤50% → 1.0 | >50% → linear decay → 0.4
  Correctness: multi-regex partial matching (0.0 – 1.0)
  
  Dimension = 10 pts | Total = 80 pts
```

### 🏷️ Multimodal Tags

| Tag | Detection | Color |
|:----|:----------|:------|
| 🔊 Audio | API test with audio | Green |
| 👁️ Vision | API test with image | Blue |
| 🎤 Speech | Name-based (tts/stt) | Purple |

## 🧪 模型能力测试

<br>

### 📊 评分

```
得分 = 基础分 × 时间系数 × 正确系数
  基础分:    每题 5 分
  时间系数:  ≤50% → 1.0 | >50% → 线性衰减 → 0.4
  正确系数:  多正则部分匹配（0.0 – 1.0）
  
  每维度 = 10 分 | 总分 = 80 分
```

### 🏷️ 多模态标签

| 标签 | 检测方式 | 颜色 |
|:-----|:---------|:-----|
